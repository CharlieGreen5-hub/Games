# Console Games!
This is a collection of Blackjack, Hangman and Minesweeper, playable in console!

## How to play
Download the latest .zip folder, and unzip it. Then run the executable file of the game you'd like to play!

All further rules and instructions can be found while running the game!

### Blackjack
Make sure that the money.txt file is in the same location as the executable, as this is necessary for the game to function properly! Change your money values at any time by changing the value in the .txt file.
